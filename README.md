# coQuest: Climate Challenge - Project Overview

This is an educational environmental awareness game built with Next.js and React.  
The game aims to educate players about climate and environmental topics while keeping them engaged through gamification elements like scores, achievements, and power-ups.

---

## Core Features

### 1. Two Main Game Levels
- **Underground Level**: Science-based quiz questions about climate, energy, water, and biodiversity  
- **Environmental Level**: Additional environmental challenges

### 2. Game Flow
- Intro screen where player enters their name  
- Progress through quiz-based levels  
- Final scoreboard showing results  

### 3. Admin Panel (password protected)
- Manage quiz questions (add/edit/delete)  
- Customize game settings (timer duration, max questions, hints)  
- Change background music  
- Toggle navigation bar visibility  

### 4. Extensions System
- **Achievements**: Unlock rewards for accomplishments  
- **Power-ups**: Special abilities players can purchase with points  
- **Mini-games**: Memory, Reaction, and Puzzle games for bonus points  

### 5. UI Features
- Navigation bar showing game name, score, and player profile  
- Background music toggle  
- Dark theme with gradient backgrounds  
- Responsive design  

---

## Tech Stack
- Next.js App Router  
- TypeScript  
- Tailwind CSS  
- shadcn/ui components
